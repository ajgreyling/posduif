# Posduif Web Application

Flutter web application for managing users and messaging.

## Structure

- `lib/core/` - Core functionality (API, auth)
- `lib/features/` - Feature modules (auth, enrollment, messaging)
- `lib/shared/` - Shared utilities and widgets

## Development

```bash
# Install dependencies
flutter pub get

# Run in Chrome
flutter run -d chrome

# Build for production
flutter build web

# Run tests
flutter test
```
